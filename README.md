# Squarage Admin Dashboard

A modern admin dashboard for Squarage company management.

## Features
- Todo list with drag-and-drop, subtasks, notes, filters
- Sales tracker with product/collection management
- Calendar with month/week/day views (desktop + mobile)
- Notes with deep link sharing
- Quick links with drag-and-drop reordering
- Expense tracking with tags
- Email subscriber management + email sending
- Push notifications (browser + iOS PWA)
- Mobile PWA with native-like experience
- Glassmorphism design system

## Tech Stack
- Next.js 15 (App Router)
- React 19
- TypeScript
- Neon PostgreSQL (Serverless)
- Drizzle ORM
- Tailwind CSS
- Zustand 5 (with store factory pattern)
- Playwright (E2E testing)
- NextAuth.js v4
